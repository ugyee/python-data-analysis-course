# Verification Checklist

## 首页内容优化

- [ ] Banner区域包含2-3行价值主张文案
- [ ] Banner区域有数据分析示意图或SVG占位
- [ ] 课程概览模块包含目标学员说明
- [ ] 课程概览模块包含前置知识要求
- [ ] 课程概览模块包含学习方式说明
- [ ] 课程概览模块包含课程总时长
- [ ] 页面底部有讲师/机构介绍区域
- [ ] 每个课程卡片有"课程大纲"按钮
- [ ] 点击"课程大纲"按钮可展开3-5个核心知识点

## 导航栏

- [ ] 导航栏包含"首页"链接
- [ ] 导航栏包含"所有课程"链接
- [ ] 导航栏包含"学习路线"链接

## 课程详情页

- [ ] 每个课程有独立路由（如/courses/data-cleaning）

- [ ] 详情页有技术栈说明（Pandas/Matplotlib等）
- [ ] 详情页有完整学习大纲（按模块划分）
- [ ] 详情页有预期学习产出说明
- [ ] 详情页有"开始项目"按钮（显示"即将开放"）
- [ ] 课程卡片有"查看详情"按钮

## 学习支持功能
